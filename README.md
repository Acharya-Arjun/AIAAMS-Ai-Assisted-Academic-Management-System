# 🎓 AI Assisted Academic Management System (AIAAMS)

> **Modernizing educational administration with intelligent analytics and a seamless user experience.**

The **AI Assisted Academic Management System (AIAAMS)** is a comprehensive, full-stack platform engineered to streamline institutional workflows. Moving beyond traditional management software, AIAAMS integrates intelligent data analysis to provide actionable insights for administrators, empower faculty with automated tracking tools, and offer students an intuitive portal for their academic journey.

---

## ✨ Core Features

### 🛡️ Admin Module (The Command Center)
* **Institutional Management:** Centralized control over students, faculty, and departments.
* **Curriculum Structuring:** Dynamic course, subject, branch, and semester management.
* **Intelligent Monitoring:** High-level dashboard analytics and reporting on institutional health.
* **Performance Tracking:** Real-time oversight of attendance and academic metrics.

### 👨‍🏫 Faculty Module
* **Course Administration:** Streamlined assignment creation and study material distribution.
* **Risk Analysis:** AI-assisted tracking of student attendance to identify at-risk individuals early.
* **Evaluation Portal:** Efficient grading systems for assignments and quizzes.
* **Actionable Insights:** Specialized analytics to gauge classroom performance trends.

### 🎒 Student Module
* **Centralized Hub:** Instant access to course materials, upcoming assignments, and grades.
* **Digital Submissions:** Secure portal for uploading assignment files.
* **Progress Tracking:** Personalized dashboards to monitor attendance and overall academic standing.

### 🧠 AI-Assisted Engine
* **Smart Analytics:** Transforms raw academic data into readable performance analyses.
* **Decision Support:** Provides data-driven recommendations for faculty and administration.
* **Predictive Monitoring:** Enhances traditional tracking with intelligent risk assessment algorithms.

---

## 🛠️ Technical Architecture

Built with a focus on high performance, scalability, and a premium UI/UX utilizing modern fluid animations.

**Frontend:**
* ⚛️ **React.js** & **JavaScript**
* 🎨 **HTML5 / CSS3**
* 💫 **Framer Motion** (For smooth, interactive UI transitions)
* 📡 **Axios** (API requests)

**Backend & AI Engine:**
* 🟢 **Node.js** & **Express.js** (Core server logic)
* 🐍 **Python** & **FastAPI** (AI processing and analytics integration)
* 🔌 **RESTful APIs**

**Database:**
* 🗄️ **MySQL** (Relational data structuring)

# Installation

## Clone Repository

```bash
git clone https://github.com/Acharya-Arjun/AIAAM-Sai-assisted-academic-management-system.git
```

## Move Project to Working Directory

For XAMPP users:

```bash
C:/xampp/htdocs/
```

## Install Frontend Dependencies

```bash
npm install
```

## Start Backend Services

Start:

* Apache
* MySQL

using XAMPP Control Panel.

## Import Database

1. Open phpMyAdmin
2. Create a database
3. Import `ams_db.sql`

## Start Frontend

```bash
npm run dev
```
